# 国标麻将实时计分系统

一个功能完善的国标麻将计分系统，支持16盘制比赛的全流程管理，包括实时计分、座位轮换、成绩统计等功能。

## 核心功能

### 1. 用户账户系统

**用户登录**
- 用户通过专属用户代码登录系统
- 首次登录自动创建账户
- 支持退出登录和重新登录
- 所有比赛数据与用户账户绑定，可在任何设备上访问

**数据隔离**
- 每个用户只能查看和管理自己创建的比赛
- 用户数据完全独立，互不干扰

### 2. 比赛创建与管理

**创建新比赛**
- 支持自定义比赛名称（可选）
- 为四位选手分别输入姓名（东、南、西、北）
- 自动创建16盘制的国标麻将比赛
- 比赛分为4轮，每轮4盘，共16盘

**比赛初始设置**
- 初始座位：A选手坐东位，B选手坐南位，C选手坐西位，D选手坐北位
- 所有选手初始分数为0分
- 自动追踪当前轮次和盘数

### 3. 实时计分功能

**和牌计分**
- 和牌选手点击自己面前的"和"按钮
- 选择和牌方式：
  - **自摸**：三家共同支付，系统自动计算每家扣分
  - **点炮**：选择点炮方，点炮方支付全部分数
- 输入番数或基本分数，系统自动计算分数变化
- 分数计算包含基本分和固定分（根据国标规则）

**分数计算规则**
- 自摸：和牌者得 `基本分 × 3 + 24分`，其他三家各付 `基本分 + 8分`
- 点炮：和牌者得 `基本分 + 24分`，点炮方付 `基本分 + 8分`，其他两家各付 `8分`

**荒庄处理**
- 当一局无人和牌时，点击"荒庄"按钮
- 荒庄情况下所有选手分数不变
- 系统自动记录荒庄局，计入比赛进度

**实时更新**
- 每局结束后分数立即更新
- 分数累计显示，方便查看当前排名
- 显示每位选手的累计得分

### 4. 座位轮换系统

**自动轮换规则**
- 每4盘为一轮，完成一轮后自动进行座位轮换
- 轮换规则：东→南→北→西→东（顺时针三个位置）
- 轮换示例：
  - 第1-4盘：A东、B南、C西、D北
  - 第5-8盘：A南、B东、C北、D西
  - 第9-12盘：A北、B西、C东、D南
  - 第13-16盘：A西、B北、C南、D东

**轮次显示**
- 显示当前轮次（东、南、西、北）
- 显示当前盘数（一、二、三、四）
- 显示总进度（如：第5盘/共16盘）
- 方位指示器会随当前盘数自动旋转

### 5. 裁判判罚功能

**判罚管理**
- 点击"判罚"按钮打开判罚界面
- 可为每位选手单独输入罚分或加分
- 支持正分（加分）和负分（扣分）
- 罚分立即生效，计入选手总分

**判罚应用场景**
- 错和处理：对错误和牌的选手进行罚分
- 诈和处理：对诈和选手进行罚分
- 违规处理：对各种违规行为进行判罚
- 补偿处理：对受到影响的选手进行加分补偿

**判罚记录**
- 判罚分数单独记录，不与正常得分混淆
- 在比赛详情中可查看判罚记录
- 判罚只能在比赛进行中设置，成绩确认后不可修改

### 6. 比赛修改功能

**撤销上一盘**
- 点击"修改"按钮可撤销上一盘的计分记录
- 撤销后盘数和轮次自动回退
- 座位信息自动恢复到撤销前的状态
- 分数重新计算，确保数据准确性

**修改限制**
- 只能撤销上一盘的记录
- 不能撤销第一盘（因为没有之前的记录）
- 需要确认操作，防止误操作

### 7. 比赛结束与确认

**正常结束**
- 打完16盘后自动进入成绩确认模式
- 四位选手需要依次点击自己的确认按钮
- 所有选手确认后，比赛正式结束
- 自动保存比赛结果和标准分

**提前结束**
- 点击"结束"按钮可提前结束比赛
- 同样需要四位选手确认成绩
- 提前结束的比赛也会被标记为已完成
- 所有已打完的盘数都会被保存

**成绩确认系统**
- 每位选手独立确认，互不影响
- 显示已确认和未确认的选手
- 确认后不可撤销（除非重新开始）
- 只有确认后的比赛才会计入统计

**标准分计算**
- 根据最终排名自动计算标准分
- 标准分规则：第1名 = 4分，第2名 = 2分，第3名 = 1分，第4名 = 0分
- 如果有并列排名，将对应名次的标准分平均分配
- 标准分用于长期战绩统计

### 8. 比赛历史管理

**查看历史记录**
- 点击主界面的"比赛历史"按钮
- 显示所有创建过的比赛列表
- 按时间倒序排列，最新的在最前面
- 进行中的比赛优先显示

**比赛状态显示**
- **进行中**：显示当前进度（如：第5盘/共16盘）
- **已完成**：表示比赛已打完并确认成绩
- **已结束**：表示比赛提前结束并确认成绩

**比赛信息展示**
- 比赛名称（如果有）
- 创建时间
- 四位选手姓名
- 当前比赛状态和进度

**继续未完成的比赛**
- 对于进行中的比赛，点击"继续"按钮
- 系统自动恢复：
  - 当前轮次和盘数
  - 所有选手的分数
  - 当前座位信息
  - 所有历史计分记录
- 可以从中断的地方无缝继续比赛

**删除比赛**
- 点击删除按钮可删除不需要的比赛
- 删除前需要确认操作
- 删除操作不可恢复，请谨慎使用
- 删除会同时清理所有相关数据（计分记录、选手信息等）

### 9. 比赛详情查看

**查看完整详情**
- 点击比赛卡片进入详情页面
- 在比赛进行中，点击"明细"按钮可查看当前比赛详情

**详情信息包括**
- 比赛基本信息：名称、创建时间、状态
- 四位选手的最终成绩和排名
- 每一盘的详细记录：
  - 盘数和轮次（如：东一局、南二局）
  - 和牌者及其方位
  - 点炮者或自摸标识
  - 基本分数
  - 四家分数变化
- 裁判判罚记录（如果有）

**详情页面功能**
- 按盘数顺序显示所有计分记录
- 清晰展示每局的得分情况
- 支持返回历史列表
- 支持返回主界面

### 10. 成绩统计功能

**个人战绩统计**
- 点击主界面的"成绩统计"按钮
- 统计所有已完成比赛中的玩家数据
- 只统计本用户创建的已完成比赛

**统计指标**
- **选手姓名**：参与过比赛的所有选手
- **参赛场次**：该选手参加的比赛总数
- **总标准分**：所有比赛的标准分累计
- **平均标准分**：总标准分 ÷ 参赛场次

**统计功能**
- 支持按不同指标排序：
  - 参赛场次
  - 总标准分
  - 平均标准分
- 支持升序和降序切换
- 数据实时更新
- 默认按总标准分降序排列

**统计用途**
- 查看长期战绩表现
- 了解选手实力排名
- 分析选手稳定性（通过平均分）
- 确定常驻选手（通过参赛场次）

### 11. 管理员功能

**管理员访问**
- 特定用户（用户代码为"micken"）可以访问管理员页面
- 主界面会显示"管理"按钮（仅管理员可见）

**管理员统计数据**
- 查看所有用户的使用统计：
  - 用户ID（用户代码）
  - 注册时间
  - 创建的比赛总数
  - 记录的盘数总数

- 按注册时间倒序显示
- 数据自动刷新

**管理员用途**
- 了解系统使用情况
- 监控用户活跃度
- 查看系统整体数据
- 进行用户支持和问题排查

## 比赛规则说明

**16盘制说明**
- 一场完整比赛共16盘
- 分为4轮，每轮4盘
- 每轮结束后进行座位轮换

**座位轮换说明**
- 选手身份（A、B、C、D）保持不变
- 方位（东、南、西、北）按规则轮换
- 分数和姓名跟随选手身份，不跟随方位

**分数计算说明**
- 包含基本分（根据番数）和固定分（8分、24分）
- 自摸和点炮的计算规则不同
- 荒庄不影响分数，直接进入下一盘

**成绩确认说明**
- 比赛结束后必须确认成绩
- 四位选手都要确认
- 确认后的比赛才会计入统计
- 确保比赛结果的公正性和准确性

## 使用建议

1. **比赛前准备**
   - 确认所有选手姓名无误
   - 可以输入有意义的比赛名称便于后续查找
   - 确保所有选手了解计分规则

2. **比赛中注意**
   - 每盘及时录入分数，避免遗忘
   - 如有错误立即使用撤销功能修正
   - 遇到特殊情况使用判罚功能处理

3. **比赛后处理**
   - 确保所有选手确认成绩
   - 可以查看详情核对每盘记录
   - 定期查看成绩统计了解长期表现
